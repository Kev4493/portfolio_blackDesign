Kanban Board

JOIN is a WebApp, inspired by the popular Kanban System for project- and personal management. It implements drag and drop lists, the creation of user boards and a user dashboard. The data are stored in a JSON array with a PHP-based script on an FTP server.

- VanillaJS

- HTML & CSS
